# Projeto Teste Ímpar

Esse projeto foi criado para teste de uma vaga para a empresa Ímpar

## Como rodar?

Clone o projeto com:
`git clone https://github.com/fernandopascoal/teste_impar.git`

instale as dependencias com:
`yarn`

rode o projeto com:
`npm start`
